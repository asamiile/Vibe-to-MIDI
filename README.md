# Vibe-to-MIDI

Vibe-to-MIDI is a make-first music learning and production assistant.

The product goal is to help users turn a vibe, movement, or sound image into DAW-ready musical choices, then learn why those choices work.

## AI Agents

- Use this in the VS Code Codex extension input:

```text
/research-feature <feature idea or research goal>
```

Example:

```text
/research-feature DAWで使う音色選択を直感的に支援するMVP
```

## Research

Research notebooks, charts, and market analysis are kept under `research/`.
See [research/README.md](research/README.md) for feature research scaffolding, environment setup, Jupyter commands, and research rules.
