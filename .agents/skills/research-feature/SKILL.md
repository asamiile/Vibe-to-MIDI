# Research Feature

Use this skill to run feature research before implementation for Vibe-to-MIDI.

Read and follow the shared command spec at `.agents/commands/research-feature.md`.

Treat the user's request as the feature idea or research goal. If the feature idea is missing, ask for it before proceeding.

Produce the outputs required by the shared spec, including the research directory, notebook, figures when useful, key finding, MVP recommendation, risks, and notebook execution status.
